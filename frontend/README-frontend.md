## Frontend (inventory-mvp)

Local development:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:
- Uses Vite + React + Tailwind.
- Routing is configured in `src/App.jsx` with `react-router-dom`.
- Place static assets in `src/assets`.
- Add reusable hooks in `src/hooks` and utilities in `src/utils`.
